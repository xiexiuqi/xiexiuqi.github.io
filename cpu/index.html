处理器概览

Intel Xeon[1]

英特尔至强处理器，x86 架构数据中心领域的主流选择。涵盖 Scalable、Max、W 等多个产品系列。

x86 架构 intel.com →

AMD EPYC[2]

AMD 霄龙处理器，基于 Zen 架构的 x86 服务器处理器，以高核心数和大缓存设计著称。

x86 架构 amd.com →

AWS Graviton[3]

亚马逊云科技自研 ARM 架构服务器处理器，专为云原生应用优化，广泛应用于 AWS 云服务。

ARM 架构 aws.amazon.com →

Ampere Altra[4]

Ampere Computing 推出的 ARM 架构云原生处理器，专为云计算和边缘计算场景设计。

ARM 架构 amperecomputing.com →

华为鲲鹏 (Kunpeng)[5]

华为基于 ARM 架构的服务器处理器，鲲鹏 920 系列是数据中心主力产品，支持国产替代。

ARM 架构 hikunpeng.com →

海光 (Hygon)[6]

海光信息 x86 架构处理器，兼容 x86 生态，面向服务器和数据中心市场，支持国产替代。

x86 架构 hygon.cn →

飞腾 (Phytium)[7]

飞腾处理器基于 ARM 架构，腾云 S 系列面向服务器和数据中心应用，支持国产替代。

ARM 架构 phytium.com.cn →

龙芯 (Loongson)[8]

龙芯处理器采用自主 LoongArch 架构，3C5000、3D5000 系列面向服务器领域，完全自主可控。

LoongArch loongson.cn →

兆芯 (Zhaoxin)[9]

兆芯 x86 架构处理器，开胜 KH-40000 系列面向服务器和高端嵌入式应用，支持国产替代。

x86 架构 zhaoxin.com →
